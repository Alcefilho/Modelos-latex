Beamer Beispiel 9

Erstellen eines Handout
Im ersten Beispiel wird Hilfe der Dokumentenklassen Option handout das Handout erstellt. Dies hat allerdings den Nachteil dass die Framegröße und das gänige DIN A 4 Format nicht übereinstimmen, d.h. ein Frame in Beamer etwa 13cm auf 10cm als Format hat.
Im zweiten Beispiel wird das Ergebniss des ersten mit Hilfe von pdfpages in eine 4 auf 1 Version umgeformt. Wer die Skripte pdfnup bzw. psnup zur Verfügung hat, erhält das gleiche Ergebnis.
Im dritten und letzten Beispiel wurde der Article Mode der Beamer Class benützt um eine Handout zu erstellen. Hier besteht die Möglichkeit neben dem Text der Präsentation zusätzlichen Text einzubringen.
handout-Option PdfpagesBeamer article

Beamer Beispiel 10

Latex Beamer class in bunt, oder wie lassen sich die Farben in der Beamer Class ändern.
Im ersten Block findet sich ein Übersicht der Farben die automatisch bei der Beamer Class verfügbar sind und der Befehl mit dem die Farbe geändert werden kann. Im zweiten eine Übersicht welche Farben durch die Verwendung der Option dvipsnames von xcolor zur Verfügung gestellt werden. Der dritte Block bezieht sich auf das Paket BeamerColor.sty welches mehrere hundert Farbtöne zur Verwendung in der Beamer Class bereitstellt.

Beamer Beispiel 11

Neben dem Ändern von Farben, kann mit Hilfe der Color Themes die Stärke der Farben beeinflusst werden. Dabei wird in drei verschiedene Color Themes bei der Beamer Class unterschieden: Inner Color Themes, für Elemente wie z.B. Blöcke usw.; Outer Color Themes, für den Titel usw.; und Complete Color Themes bei denen neben der Farbe auch die Stärke festgelgt wird.
Inner Color Themes Outer Color Themes Complete Color Themes
Inner Color Themes kurz(pdf)  Outer Color Themes kurz (pdf)   Complete Color Themes kurz (pdf)  
Usetheme Warsaw mit Inner Color Theme Lily (pdf)  Usetheme Warsaw mit Outer Color Theme Dolphin (pdf)   Complete Color Theme [overlystylish] Albatross (pdf)  
Usetheme Warsaw mit Inner Color Theme Lily (tex)   Usetheme Warsaw mit Outer Color Theme Dolphin (tex)  Complete Color Theme [overlystylish] Albatross (tex)
