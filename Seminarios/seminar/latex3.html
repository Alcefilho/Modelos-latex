Presentations using pdfLaTeX


In this workshop, we have discussed and shown examples of the following ways of creating presentation documents in LaTeX:
  • the use of "seminar" document class in LaTeX
  • the use of "foils" document class with the FoilTeX package
  • hyperreferencing in PDF with the hyperref package
  • creating powerpoint-like flashy effects with the ppower4 package
