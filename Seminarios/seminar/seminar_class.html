Presentation using the LaTeX seminar class



  • To create a simple presentation document using LATEX, we can use the seminar class. The syntax is as follows: 

    \documentclass[landscape]{seminar}
...
\begin{document}
\begin{slide}
...
\newslide
...
\end{slide}
\begin{slide}
...
\newslide
...
\end{slide}
\end{document}

    Note that we have used the landscape mode for the slides. One can substitute "landscape" by "portrait" if the portrait mode is desired for the entire document. Also note the use of slide environment within the document, and the use of the \newslide command to strongly control page breaks as it is important to control the specific material on each page.


  • We can type mathematical formulas and include graphics as in other classes of LaTeX documents. For the latter, remember to add \usepackage{graphicx} in the preamble.

    For example, to insert a plot named "mypic.ps" or "mypic.pdf" in a framebox, use the command:

        \fbox{\includegraphics[height=0.8\textheight]{mypic}}

    Note that in the above command, we need not specify the file extension of the image file. If we use latex file.tex, it will automatically search for .ps file to include. Likewise, if we use pdflatex file.tex, the .pdf files will be searched.


  • How to temporarily switch between landscape mode and portrait mode?

    The syntax is 

    \begin{slide*}
 :
\newslide
 :
\end{slide*}

    The asterisk is necessary for switching from the predefined mode, say, landscape mode, to the portrait mode, and vice versa.


  • If you have both landscape and portrait slides in your file, there are two ways to print the slides:

    • If your dvi driver supports rotation, then you can print all the slides at once. For example, by including the package semrot in the preamble, you can directly print all the slides at once.
    • You can first print your landscape slides by putting the command 
        \landscapeonly
      in the preamble, and then print the portrait slides by inserting instead the command 
        \portraitonly
      and switching to the portrait style option in the documentclass statement.


  • To create header, footer and ruler in your slides, use the fancyhdr package:

    Add to the preamble

        \usepackage{fancyhdr}

    Commands available from the package:

    • \pagestyle{fancy}
    • \fancyhf{} to delete current setting for header and footer
    • \lhead{...} for left header
    • \chead{...} for center header
    • \rhead{...} for right header
    • \renewcommand{\headrulewidth}{0.5pt}
    • \lfoot{...} for left footer
    • \cfoot{...} for center footer
    • \rfoot{...} for right footer
    • \renewcommand{\footrulewidth}{0pt}

    Alternatively, we can use

    • \pagestyle{fancy}
    • \fancyhf{}
    • \fancyhead[L]{...}
    • \fancyhead[C]{...}
    • \fancyhead[C]{...}
    • \renewcommand{\headrulewidth}{0.5pt}
    • \fancyfoot[L]{...}
    • \fancyfoot[C]{...}
    • \fancyfoot[R]{...}
    • \renewcommand{\footrulewidth}{0pt}
