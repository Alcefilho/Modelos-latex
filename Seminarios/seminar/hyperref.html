Hyperreferencing in the FoilTeX Class



To do hyperreferencing within the presentation document file, use the hyperref package:

  • The package hyperref derives from, and builds on, the work of the HyperTEX project, described at http://xxx.lanl.gov/hypertex/. (xxx is not a typo!)


  • It extends the functionality of all the LATEX cross-referencing commands (including the table of contents, bibliographies etc).


  • It also provides new commands to allow the user to write ad hoc hypertext links, including those to external documents and URLs.

Syntax:

  • \href{URL}{text}

    Example: \href{http://www.stat.ubc.ca}{Stat Dept., UBC}

    The above line of code produces a link entitled "Stat Dept., UBC" in the output file (either ps or pdf format). Upon clicking the link, a web brower will be brought up and the page indicated by the link will be loaded.

  • \hyperlink{name}{text} and \hypertarget{name}{text}

    Example: Suppose we want to create a link to the slide with the heading "Color Package". When we create that particular slide, do

       \foilhead[-0.5in]{\hypertarget{mycolor}{Color Package}}

    and at the place where the link is to be added, do

        \hyperlink{mycolor}{Color Package}

    The "mycolor" name is used for referencing, while "Color Package" will be shown as the link in the output pdf file.



To see how hyperreferencing works, take a look at an example presentation document (using the "foils" document class)
